Installer l'APK sur Android

Le navigateur ne peut pas installer directement. Suivez ces étapes :
1
Téléchargez l'APK reconstruit
2
Transférez-le sur votre appareil (USB, cloud…)
3
Activez Sources inconnues dans Paramètres → Sécurité
4
Ouvrez le fichier APK sur l'appareil
Astuce : adb install fichier.apk pour une install via USB.
